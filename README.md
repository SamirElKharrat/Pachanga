- Título de la idea de proyecto: La Pachanga 
- Breve descripción del proyecto
Esto es un proyecto personal que llevo 3 años llevando a mano, la idea sería pasarlo a web y si puedo a movil. Es una especie de predicciones de partidos del juego League of Legends, enfocado en el competitivo del mismo. Nosotros votamos quien creemos que va a ganar y vamos ganando puntos según lo que acertamos, tambien hay preguntas.... la idea es para pasarselo bien y conectar mas a la gente de un entorno. 
- Listado de funcionalidades principales:

1.Creacion y gestion de formularios para los votos

2.Listado tanto de las ligas, partidos, clasificaciones, ...

3.Gestión de Usuario con Roles

4.Unirse o Salirse de Ligas
